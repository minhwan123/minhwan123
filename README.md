## 🚀 대표 프로젝트

### 📄 [Multimodal PPG-Based Arrhythmia Detection](https://github.com/minhwan123/ppg-arrhythmia-detection)
PPG(광용적맥파) 신호만으로 부정맥을 탐지·구간분할하고, LLM이 근거 기반 진단 리포트를 생성하는 end-to-end 프레임워크.
**MDPI *Sensors* (SCIE) 게재 논문**의 공식 구현체입니다.
`PyTorch` `CLIP-initialized U-Net` `LLM Reporting`

### 🎭 [Cullect — 문화생활 추천 시스템](https://github.com/minhwan123/Cullect)
동국대 공개SW프로젝트, **팀장 겸 AI 추천 시스템 개발**.
사용자 프로필·행동 로그 기반 **Collaborative Filtering + XGBoost 하이브리드 추천 엔진** 설계·구현.
`FastAPI` `XGBoost` `Hybrid Recommender`

### 🎨 [CustomFace — 실시간 얼굴 캐리커처 생성기](https://github.com/minhwan123/CustomFace)
동국대 HCI 수업 팀 프로젝트.
웹캠으로 촬영한 얼굴을 실시간 분석해 랜드마크 기반으로 과장·카툰화하는 캐리커처 생성기.
