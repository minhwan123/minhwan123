## 🚀 대표 프로젝트

### 📄 [Multimodal PPG-Based Arrhythmia Detection](https://github.com/minhwan123/ppg-arrhythmia-detection)

`CLIP-initialized U-Net` `Multi-Task Learning` `LLM Reporting`

### 🎭 [Cullect — 문화생활 추천 시스템](https://github.com/minhwan123/Cullect)

`Recommender System` `Hybrid CF` `XGBoost`

### 🚗 [Car-BTI — 전국 자동차 소비 성향 분석 대시보드](https://github.com/minhwan123/Car-BTI)

`Streamlit` `Data ETL` `RAG Chatbot`
